# react-native-toast-show

Here is library to show toast in IOS && ANDROID

## Installation

```sh
npm install react-native-toast-show
```

## Usage

```js
import { ToastShowView } from "react-native-toast-show";

// ...

<ToastShowView color="tomato" />
```
